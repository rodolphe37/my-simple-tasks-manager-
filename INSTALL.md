## :microscope: Getting Started!

you just have to **clone this repository**.

---

## :satellite: Install dependencies

On the root folder:

```shell
yarn
```
or
```shell
npm i
```
---

## :rainbow: Start the App

```shell
yarn start
```
or
```shell
npm start
```


Your site starts at `http://localhost:3000`.

---


* **Enjoy!**
